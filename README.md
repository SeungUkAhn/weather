# Weather Now FE
기상청 API를 활용한 날씨예보 서비스 제작 토이프로젝트 Weather Now의 프론트엔드 프로젝트입니다. 
분단위로 변화할 수 있는 데이터를 고려하여 React를 프레임워크로 사용합니다.

# 기능
- API호출을 통한 현재 날씨, 단기 예보 화면 출력 (중기 예보 작업 중)
